# Tic Tac Toe Game

React로 구현한 클래식 틱택토(Tic-Tac-Toe) 게임입니다.

## 📋 프로젝트 소개
두 명의 플레이어가 3x3 격자판에서 번갈아가며 'X'와 'O'를 두어 승부를 겨루는 게임입니다. 게임 상태 관리와 승리 조건 체크 로직이 포함되어 있습니다.

## ✨ 기능
- **게임 플레이**: 클릭으로 'X' 또는 'O'를 놓을 수 있습니다.
- **순서 전환**: 플레이어의 턴이 자동으로 전환됩니다.
- **승리 판별**: 가로, 세로, 대각선으로 3개가 이어지면 승리자가 표시됩니다.
- **게임 초기화**: 게임을 다시 시작할 수 있는 기능이 제공됩니다.

## 🛠 기술 스택
- **React**: 게임 로직 및 UI 상태 관리
- **CSS**: 게임 보드 및 스타일링
